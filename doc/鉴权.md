# 鉴权

背景：老鱼人知识库控制台界面(管理系统)
鉴权模块，负责用户进入调试后台，监控agent系统等



## 实现思路

采用签发token(**1h**)的形式，后续用户在后台的请求都需要校验token，只颁发短时间的访问令牌，**保证安全性**。

- [ ] 接收前端输入的用户名密码 
- [ ] 密码加密后和数据库进行比对
- [ ] 若一致，向用户颁发访问令牌
  
  

## 准备

### 数据库用户表设计

```sql
CREATE TABLE users (
    -- 内部唯一ID（自增主键）
    id BIGSERIAL PRIMARY KEY,

    -- 对外唯一ID（UUID，推荐使用）
    user_uuid UUID DEFAULT gen_randouuid() NOT NULL UNIQUE,

    -- IP地址（支持IPv4和IPv6）
    ip_address INET,

    -- 最后登录IP（便于风控）
    last_login_ip INET,

    -- 其他业务字段
    username VARCHAR(50) UNIQUE,
    email VARCHAR(100) UNIQUE,
    phone VARCHAR(20) UNIQUE,
    password_hash VARCHAR(255) NOT NULL,

    -- 状态字段
    status SMALLINT DEFAULT 1,  -- 1:正常 0:禁用 -1:删除

    -- 时间戳
    created_at TIMESTAMPTZ DEFAULT NOW(),
    updated_at TIMESTAMPTZ DEFAULT NOW(),
    last_login_at TIMESTAMPTZ
);

-- 索引
CREATE INDEX idx_users_user_uuid ON users(user_uuid);
```



## 实现

1.根据上方数据库语句在/db/models中创建对应的实体

2.在routes/login.py中创建接口，register，login，logout用于实现登录登出

3.对应的业务代码在/services下实现，重点注意密码，需要加密后入库，校验
